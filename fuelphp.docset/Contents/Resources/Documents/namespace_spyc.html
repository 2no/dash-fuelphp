This class can be used to read a YAML file and convert its contents into a PHP array. It currently supports a very limited subsection of the YAML spec. Usage: $Spyc = new Spyc; $array = $Spyc->load($file); or: $array = Spyc::YAMLLoad($file); or: $array = spyc_load_file($file);
