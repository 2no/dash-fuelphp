Searches for the given variable and returns its value. Local variables will be returned before global variables. 

$value = $view->get('foo', 'bar');

If the key is not given or null, the entire data array is returned.

Fuel\Core\View::set (   $key,
  $value = null,
  $filter = null 
)

Assigns a variable by name. Assigned values will be available as a variable within the view file: 

// This value can be accessed as $foo within the view
$view->set('foo', 'my value');

You can also use an array to set several values at once: 

// Create the values $food and $beverage in the view
$view->set(array('food' => 'bread', 'beverage' => 'water'));
