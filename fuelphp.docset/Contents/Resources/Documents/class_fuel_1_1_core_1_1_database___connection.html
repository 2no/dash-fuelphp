Fuel\Core\Database_Connection::quote (   $value)

Quote a value for an SQL query. 

$db->quote(null);   // 'null'
$db->quote(10);     // 10
$db->quote('fred'); // 'fred'

Objects passed to this function will be converted to strings. [Database_Expression] objects will use the value of the expression. [Database_Query] objects will be compiled and converted to a sub-query. All other objects will be converted using the __toString method.

Fuel\Core\Database_Connection::quote_identifier (   $value)

Quote a database identifier, such as a column name. Adds the table prefix to the identifier if a table name is present. 

$column = $db->quote_identifier($column);

You can also use SQL methods within identifiers. 

// The value of "column" will be quoted
$column = $db->quote_identifier('COUNT("column")');

Objects passed to this function will be converted to strings. [Database_Expression] objects will use the value of the expression. [Database_Query] objects will be compiled and converted to a sub-query. All other objects will be converted using the __toString method.

Parameters
mixed$valueany identifier
Returns
