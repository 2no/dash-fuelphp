Detailed Description

Implements NestedSets (http://en.wikipedia.org/wiki/Nested_set_model)

Some design ideas borrowed from https://github.com/hubspace/fuel-nestedset (see http://fuelphp.com/forums/discussion/12206/fuel-nested-sets)
